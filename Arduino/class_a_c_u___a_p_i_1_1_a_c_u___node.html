Detailed Description

Abstract Base of the ACU PCB within the ARU.

Within the ARU an ACU is intended to be the data hub for all sub-systems. It will need a minimum of two Packet Ports, three if dual terminal ARU.

An ACU also interfaces with a Matrix Orbital Display and a Diversity switch bank on dual terminal systems.
