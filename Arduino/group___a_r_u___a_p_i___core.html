Detailed Description

API Node and Interface Objects.

Here is the first level of personalization to make the abstract elements of the IMS_PacketsAPI_Core.h file specific to the Envistacom's ARU FW.

The language of the distributed ARU FW system is described here as a structure of Packet objects. Knowledge of the language is shared by including Packet strutures in ARU_API_Node objects which derive from the AbstractDataExecution class
